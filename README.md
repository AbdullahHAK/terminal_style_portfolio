# Abdullah Terminal Portfolio

A fully immersive, browser-based Unix-style terminal portfolio for Abdullah — built from scratch with vanilla HTML, CSS, and JavaScript.

This project is designed to inspire developers, recruiters, and terminal lovers by turning a portfolio into an interactive shell experience:

- authentic boot sequence with ASCII branding
- live typewriter command output
- command history, tab completion, and keyboard shortcuts
- clickable links, resume download, and `sudo hire` flow
- responsive layout with subtle CRT scanlines and flicker

## Why this repo is worth starring

- **Unique concept:** portfolio as a terminal emulator, not a standard website.
- **Zero dependencies:** handcrafted with plain HTML/CSS/JS.
- **Production-ready:** compatible with GitHub Pages and static hosting.
- **Engaging UX:** every interaction feels like using a real shell.

## Files

- `index.html` — terminal entry point
- `style.css` — terminal theme, responsive layout, animation
- `terminal.js` — command engine, boot sequence, input behavior
- `resume.pdf` — downloadable resume sample

## Live demo
Deploy `index.html` to GitHub Pages or Netlify for a working portfolio demo.

## Interaction tips

- `Ctrl+L` clears the terminal
- `Ctrl+C` cancels the current input
- `Tab` autocompletes commands
- `Arrow Up` / `Arrow Down` cycles command history
